# Retail Sales Exploratory Data Analysis (EDA) 📊

This project focuses on analyzing retail sales data to uncover key business insights, customer behavior patterns, and revenue trends. It was developed as an end-to-end data visualization project using Python.

## 🚀 Project Overview
The goal of this analysis is to transform raw retail transaction data into actionable insights. By using various data visualization techniques, I explored:
*   **Monthly Revenue Trends:** Understanding growth over time.
*   **Product Performance:** Identifying top-selling categories.
*   **Customer Demographics:** Analyzing sales distribution by Gender and Age.
*   **Temporal Patterns:** Finding the busiest days of the week and high-performing quarters.

## 🛠️ Tech Stack & Tools
*   **Language:** Python
*   **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
*   **Environment:** VS Code / Google Colab
*   **Dataset:** Retail Sales Transaction Data

## 📈 Key Insights from Dashboard
Based on the visualizations, here are some critical findings:
1.  **Revenue Concentration:** The **Electronics** category leads in revenue, followed closely by Clothing.
2.  **Peak Activity:** Sales show a significant spike on **Saturdays**, suggesting higher weekend consumer engagement.
3.  **Customer Insights:** **Female shoppers** contribute slightly more to the total revenue compared to male shoppers.
4.  **Quarterly Performance:** Q2 and Q4 showed the strongest performance, likely due to seasonal demand.

## 📂 Project Structure
*   `Analysis.ipynb`: The main notebook containing the data cleaning and visualization code.
*   `retail_sales.csv`: The dataset used for analysis.
*   `output/`: Folder containing  final dashboard image.
